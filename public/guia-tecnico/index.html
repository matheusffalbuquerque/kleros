🌐 Estrutura Multi-Domain

Sistema de domínios para multi-tenancy

Domínio Propósito Middleware Acesso
kleros.local Site institucional e registro setlocale Público
admin.local Painel administrativo setlocale, auth Admins/Kleros
{congregacao}.local Sistema da congregação específica dominio, auth, member.activity Membros da congregação

🔑 Fluxo de Autenticação

Como funciona o sistema de autenticação e autorização

// Middleware de autenticação por domínio
AcessarCongregacaoPeloDominio.php
├── Identifica congregação pelo subdomínio
├── Injeta congregação no container (app('congregacao'))
├── Configura contexto da sessão
└── Redireciona se congregação inativa

// Roles do sistema
Spatie Permission
├── 'kleros'  → Acesso administrativo total
├── 'admin'   → Gestão denominacional  
└── 'gestor'  → Gestão congregacional
